# 桌面时钟 · 网络校时 · 本地背景
一个简洁、优雅的桌面风格网页时钟，支持**网络时间同步**和**自定义背景图片**。打开浏览器，即可获得一个美观的时钟面板，适合放在桌面作为装饰或工作辅助

## ✨ 功能特性
- **实时时间显示** – 大号数字显示当前时间（时:分:秒），下方显示日期（年月日+星期）。
- **网络时间同步** – 点击按钮从网络获取准确的 UTC 时间，并自动转换为您的本地时区显示，消除系统时间误差。
- **本地背景上传** – 从本机选择任意图片作为时钟背景，支持一键清除背景，恢复默认深色。
- **字体规范** – 数字使用经典的 Times New Roman，日期文字使用 Adobe 宋体 Std L（或系统回退字体），视觉效果清晰典雅。
- **离线可用** – 未联网时自动使用设备本地时间，保证时钟基本功能。
- **响应式设计** – 自适应不同屏幕尺寸，手机、平板、电脑均可舒适查看。
- **状态指示** – 左下角显示当前时间源（本地/网络），点击可查看详情。

## 🚀 如何使用
1. 下载本仓库的 `index.html` 文件（或直接复制下方代码保存为 `.html` 文件）。
2. 使用现代浏览器（Chrome / Edge / Firefox）打开该文件。
3. 享受时钟：
   - 点击 **“同步网络时间”** 校准时间。
   - 点击 **“选择背景图片”** 上传本地图片作为背景。
   - 点击 **“清除背景”** 恢复默认背景。

## 🛠️ 技术实现
- **时间基准**：同步时记录网络 UTC 时间戳 + `performance.now()`，后续计算均基于此基准，避免系统时间被篡改导致的误差。
- **网络 API**：
  - 主用：`worldtimeapi.org/api/timezone/Etc/UTC`（返回 unixtime）
  - 备用：`timeapi.io/api/Time/current/zone?timeZone=UTC`（返回 dateTime）
  - 请求超时 5 秒，失败时自动降级为本地时间。
- **背景上传**：使用 `URL.createObjectURL` 生成临时 URL 设置 `body` 背景，并自动释放内存。
- **字体控制**：CSS 中通过 `.number-font` 类单独为日期中的数字指定 Times New Roman，其余文字保持 Adobe 宋体。
- **状态管理**：通过 `isNetworkSynced` 标志切换时间来源，界面实时反映。

## 📦 文件结构
- `index.html` – 单文件包含所有样式、结构和逻辑，可直接运行。

## 🔧 自定义与扩展
- **更换 API**：如需更换时间源，修改 `syncWithNetwork` 函数中的 `API_URLS` 数组即可。
- **调整字体**：修改 CSS 中 `font-family` 规则，替换为您喜欢的字体。
- **修改时间格式**：编辑 `updateDisplay` 函数中的日期/时间拼接逻辑。

## ⚠️ 注意事项
- 网络同步功能依赖外部 API，若所在网络无法访问 `worldtimeapi.org`，可自行替换为其他可用接口（如 `https://quan.suning.com/getSysTime.do` 需注意时区处理）。
- 背景图片仅保存在浏览器内存中，刷新页面后失效，这是出于安全考虑（无法持久化存储用户文件）。
- 长时间运行（数天）可能出现微小漂移，再次点击同步即可校准。
